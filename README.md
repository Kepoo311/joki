# Joki Arceus Project
Joki Arceus is a website for people to rent a jockey service

## Features

- Rent a jockey
- Customer service
- Multi level role
- Cross platform


## Installation

After you clone this project pls to this step

```bash
  composer update
make database name joki_db or you can change it from the env
  php artisan serve

```
    
## Tech Stack

**Client:** Laravel, TailwindCSS
